## 🌐 Как опубликовать веб-версию онлайн (GitHub Pages)

### Шаг 1: Создайте GitHub аккаунт
1. Перейдите на https://github.com
2. Нажмите "Sign up" и заполните форму

### Шаг 2: Создайте репозиторий
1. На главной странице нажмите "New"
2. Назовите его: `blockade-map` или любое другое имя
3. Выберите "Public" (публичный)
4. Нажмите "Create repository"

### Шаг 3: Загрузьте файл
1. На странице репозитория нажмите "Add file" → "Upload files"
2. Перетащите `index.html` в область загрузки
3. Нажмите "Commit changes"

### Шаг 4: Включите GitHub Pages
1. Перейдите в "Settings" репозитория
2. Найдите "Pages" слева
3. Выберите "Main branch" как источник
4. Нажмите "Save"

### Шаг 5: Получите ссылку
1. GitHub создаст ссылку в формате: `https://[ваш_ник].github.io/blockade-map`
2. Скопируйте эту ссылку и поделитесь с преподавателем
