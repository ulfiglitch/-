# 🗺️ Карта памяти: Блокада Ленинграда

**Интерактивное веб-приложение для исследования истории блокады Ленинграда**

## 📱 Что это?

Это полнофункциональное интерактивное веб-приложение, которое позволяет:
- 🗺️ **Исследовать события на интерактивной карте** Ленинграда
- ⏱️ **Навигировать по датам** (872 дня блокады) с помощью слайдера
- 📍 **Просматривать события** с описаниями и архивными материалами
- 📊 **Анализировать статистику** ключевых показателей
- 🎯 **Фильтровать события** по типам (военные, социальные, повседневные)

## 🎨 Что внутри приложения

### 1. **Интерактивная карта**
- Карта Ленинграда с маркерами событий
- Цветовое кодирование по типам событий:
  - 🔴 **Красные** = Военные события
  - 🔵 **Синие** = Социальные события
  - 🟠 **Оранжевые** = Повседневная жизнь
- Клик на маркер = Открыть полную информацию о событии
- Поддержка зума и панорамирования

### 2. **Слайдер временной шкалы**
- Выберите любую дату от 8 сентября 1941 до 27 января 1944
- События на карте обновляются в реальном времени
- Показывает текущую дату и прошедшие дни блокады

### 3. **Карточки событий**
- Отображает все события, соответствующие выбранной дате
- Нажмите на событие для подробной информации
- Показывает описание, дату, тип события

### 4. **Модальные окна**
- Клик на событие открывает полное описание
- Содержит источник информации
- Легко закрывается (кнопка X или клик вне окна)

### 5. **Фильтры по типам**
- **Все события** – Показать все события
- **⚔️ Военные** – Только военные действия
- **👥 Социальные** – Социальные события (голод, эвакуация и т.д.)
- **🏠 Повседневная** – Повседневная жизнь в городе

### 6. **Статистика**
- Общее количество дней блокады (872)
- Погибших (1М+)
- Количество событий на выбранную дату
- Всего документированных событий (50+)

### 7. **Навигация**
- Липкий заголовок с меню
- Ссылки на разделы: Карта, События, Статистика, О проекте
- Кнопка "О проекте" для информации о разработке

## 🔧 Как добавить собственные события

Найдите в коде раздел `const events = [...]` и добавьте объект:

```javascript
{
    id: 9,
    date: '1 февраля 1942',
    dayNumber: 146,
    title: 'Название события',
    description: 'Подробное описание события...',
    type: 'military', // или 'social', 'daily'
    emoji: '🔴', // Выберите эмодзи
    lat: 59.9311, // Широта (Ленинград ~59.93)
    lng: 30.3609, // Долгота (Ленинград ~30.36)
    source: 'Название источника'
}
```

## 🚨 Возможные проблемы и решения

### Карта не загружается
**Решение:** Убедитесь, что у вас есть интернет (для загрузки тайлов карт).

### События не появляются на карте
**Решение:** Переместите слайдер вперед. События появляются только до выбранной даты.

### Слайдер не работает
**Решение:** Используйте браузер Chrome, Firefox или Safari (не очень старые версии).

### Модальное окно не закрывается
**Решение:** Нажмите кнопку X в правом верхнем углу или кликните вне окна.
