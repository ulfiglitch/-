# Карта памяти Блокада Ленинграда

Интерактивная веб-хроника 872-дневной блокады Ленинграда (1941-1944).

## Описание проекта

Проект объединяет исторические события, географические координаты и интерактивную шкалу времени в единое приложение для изучения истории Великой Отечественной войны.

### App

1. Откройте `app/index.html` в браузере
2. Используйте слайдер для навигации по датам (0-872 дни)
3. Нажимайте на события для просмотра подробностей

### 

```bash
pip install -r backend/requirements.txt
python backend/manage.py runserver
```

## Как вносить изменения

1. Создайте ветку для своей работы
2. Вносите изменения в соответствующую папку (frontend/backend/docs)
3. Создавайте Pull Request с описанием изменений
