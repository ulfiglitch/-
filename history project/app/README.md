# Frontend — Интерфейс приложения

Пользовательский интерфейс "Карты памяти" на HTML, CSS и JavaScript.

## Файлы

### index.html
Основная структура страницы:
- `<nav>` — навигационная панель сверху
- `<div class="container">` — основной контент
- Карта Leaflet.js в `<div id="map">`
- Боковая панель с управлением временем

### style.css
Стили для всех элементов:
- Навигация (nav)
- Сетка макета (grid layout)
- Стили кнопок фильтров
- Оформление карточек событий
- Модальное окно для подробностей

### app.js
Логика приложения:
- `displayEvents(day)` — показать события за выбранный день
- `updateMap(day)` — обновить маркеры на карте
- `updateDate(day)` — обновить дату
- `filterEvents(type)` — фильтровать события
- `showModal(event)` — показать подробности события

### data.js
Данные исторических событий:
- Массив `events` с 10 ключевыми событиями блокады
- Каждое событие содержит:
  - title — название события
  - date — дата в формате YYYY-MM-DD
  - day — номер дня блокады (0-872)
  - type — тип (military, social)
  - lat/lng — координаты на карте
  - description — описание события
  - source — источник информации

## Как работает приложение

1. Когда страница загружается, `app.js` инициализирует карту Leaflet
2. При движении слайдера срабатывает обработчик `input`
3. Функция `updateDate(day)` обновляет все элементы для выбранного дня
4. События отфильтровываются по типу и дню
5. На карту добавляются маркеры для событий до выбранного дня
6. При клике на событие открывается модальное окно

## Модули и зависимости

- **Leaflet.js** — библиотека для интерактивных карт (CDN)
- OpenStreetMap — база карт

## Как добавить новое событие

1. Откройте `data.js`
2. Добавьте объект в массив `events`:

```javascript
{
    id: 11,
    title: "Название события",
    date: "YYYY-MM-DD",
    day: 750,
    type: "military",
    lat: 59.930,
    lng: 30.362,
    description: "Описание события",
    source: "Источник информации"
}
```

## Как изменить стили

1. Откройте `style.css`
2. Найдите нужный селектор (например, `.event-item`)
3. Измените свойства CSS

## Быстрая справка по селекторам

- `#map` — карта
- `.sidebar` — боковая панель
- `.event-item` — карточка события
- `.filters button` — кнопки фильтров
- `.modal` — модальное окно

## Тестирование

Откройте `index.html` в браузере:
1. Двигайте слайдер влево-вправо
2. Нажимайте на события
3. Тестируйте фильтры
